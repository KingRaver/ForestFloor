# Starter Kit

This folder ships default one-shot WAV samples for first launch playback.

Track mapping:
- Track 1: `kick.wav`
- Track 2: `snare.wav`
- Track 3: `clap.wav`
- Track 4: `hat_closed.wav`
- Track 5: `hat_open.wav`
- Track 6: `tom_low.wav`
- Track 7: `tom_high.wav`
- Track 8: `perc.wav`

`default.ffproject` is a ready-to-play 16-step session using these samples.
